## TAN
## About the function
This function is used to calculate the tangent of a time series. **Note!** The
values of the input are in radians.

## Syntax
- TAN(t)

| # | Type | Description |
|---|---|---|
| 1 | t | Time series |

## Example
`Temperature_hour_VV = @TAN(@t('Temperature_hour_raw'))`
